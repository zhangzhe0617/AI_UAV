使用优化器

优化器是编译Keras模型所需的两个参数之一：

from keras import optimizers

model = Sequential()
model.add(Dense(64, kernel_initializer='uniform', input_shape=(10,)))
model.add(Activation('softmax'))

sgd = optimizers.SGD(lr=0.01, decay=1e-6, momentum=0.9, nesterov=True)
model.compile(loss='mean_squared_error', optimizer=sgd)

您可以在传递优化器之前将其实例化model.compile()，如上例所示，或者您可以通过其名称来调用它。在后一种情况下，将使用优化程序的默认参数。

# pass optimizer by name: default parameters will be used
model.compile(loss='mean_squared_error', optimizer='sgd')

所有Keras优化器通用的参数

的参数clipnorm和clipvalue可以与所有优化可以用来控制限幅梯度：

from keras import optimizers

# All parameter gradients will be clipped to
# a maximum norm of 1.
sgd = optimizers.SGD(lr=0.01, clipnorm=1.)

from keras import optimizers

# All parameter gradients will be clipped to
# a maximum value of 0.5 and
# a minimum value of -0.5.
sgd = optimizers.SGD(lr=0.01, clipvalue=0.5)

SGD

keras.optimizers.SGD(lr=0.01, momentum=0.0, decay=0.0, nesterov=False)

随机梯度下降优化器。

包括对动量，学习率衰减和Nesterov势头的支持。

参数

  • lr：float> = 0.学习率。
  • 动量：float> = 0.参数可以在相关方向上加速SGD并抑制振荡。
  • 衰减：浮动> = 0.每次更新时学习率下降。
  • nesterov：布尔值。是否适用Nesterov势头。

[资源]

RMSprop

keras.optimizers.RMSprop(lr=0.001, rho=0.9, epsilon=None, decay=0.0)

RMSProp优化器。

建议将此优化器的参数保留为默认值（学习速率除外，可以自由调整）。

这种优化器通常是递归神经网络的不错选择。

参数

  • lr：float> = 0.学习率。
  • rho：float> = 0。
  • epsilon：float> = 0.模糊因子。如果None，默认为K.epsilon()。
  • 衰减：浮动> = 0.每次更新时学习率下降。

参考

  • [rmsprop：将梯度除以最近幅度的平均值]（http://www.cs.toronto.edu/~tijmen/csc321/slides/lecture_slides_lec6.pdf）

[资源]

Adagrad

keras.optimizers.Adagrad(lr=0.01, epsilon=None, decay=0.0)

Adagrad优化器。

Adagrad是一种具有参数特定学习率的优化器，它根据参数在训练期间的更新频率进行调整。参数接收的更新越多，更新越小。

建议将此优化程序的参数保留为默认值。

参数

  • lr：float> = 0.初始学习率。
  • epsilon：float> = 0.如果None，默认为K.epsilon()。
  • 衰减：浮动> = 0.每次更新时学习率下降。

参考

  • 在线学习和随机优化的自适应子梯度方法

[资源]

Adadelta

keras.optimizers.Adadelta(lr=1.0, rho=0.95, epsilon=None, decay=0.0)

Adadelta优化器。

Adadelta是Adagrad的一个更强大的扩展，它根据渐变更新的移动窗口调整学习速率，而不是累积所有过去的渐变。这样，即使进行了许多更新，Adadelta仍在继续学习。与Adagrad相比，在Adadelta的原始版本中，您无需设置初始学习率。在此版本中，可以设置初始学习速率和衰减因子，与大多数其他Keras优化器一样。

建议将此优化程序的参数保留为默认值。

参数

  • lr：float> = 0.初始学习率，默认为1.建议将其保留为默认值。
  • rho：float> = 0. Adadelta衰减因子，对应于每个时间步长保持的梯度分数。
  • epsilon：float> = 0.模糊因子。如果None，默认为K.epsilon()。
  • 衰变：浮动> = 0.初始学习率衰减。

参考

  • [Adadelta - 一种自适应学习率方法]（https://arxiv.org/abs/1212.5701）

[资源]

亚当

keras.optimizers.Adam(lr=0.001, beta_1=0.9, beta_2=0.999, epsilon=None, decay=0.0, amsgrad=False)

亚当优化器。

默认参数遵循原始论文中提供的参数。

参数

  • lr：float> = 0.学习率。
  • beta_1：float，0 <beta <1。一般接近1。
  • beta_2：float，0 <beta <1。一般接近1。
  • epsilon：float> = 0.模糊因子。如果None，默认为K.epsilon()。
  • 衰减：浮动> = 0.每次更新时学习率下降。
  • amsgrad：布尔值。是否应用“论亚当与超越的融合”一文来应用该算法的AMSGrad变体。

参考

  • [亚当 - 随机优化的方法]（https://arxiv.org/abs/1412.6980v8）
  • [关于亚当及其他人的融合]（https://openreview.net/forum?id=ryQu7f-RZ）

[资源]

Adamax

keras.optimizers.Adamax(lr=0.002, beta_1=0.9, beta_2=0.999, epsilon=None, decay=0.0)

来自Adam paper第7节的Adamax优化器。

它是基于无限规范的亚当的变体。默认参数遵循论文中提供的参数。

参数

  • lr：float> = 0.学习率。
  • beta_1 / beta_2：浮点数，0 <beta <1。一般接近1。
  • epsilon：float> = 0.模糊因子。如果None，默认为K.epsilon()。
  • 衰减：浮动> = 0.每次更新时学习率下降。

参考

  • [亚当 - 随机优化的方法]（https://arxiv.org/abs/1412.6980v8）

[资源]

那达慕

keras.optimizers.Nadam(lr=0.002, beta_1=0.9, beta_2=0.999, epsilon=None, schedule_decay=0.004)

Nesterov Adam优化器。

就像亚当基本上是具有动力的RMSprop一样，Nadam是具有Nesterov势头的Adam RMSprop。

默认参数遵循论文中提供的参数。建议将此优化程序的参数保留为默认值。

参数

  • lr：float> = 0.学习率。
  • beta_1 / beta_2：浮点数，0 <beta <1。一般接近1。
  • epsilon：float> = 0.模糊因子。如果None，默认为K.epsilon()。

参考

  • 纳达姆报道
  • [关于初始化和深度学习动力的重要性]（http://www.cs.toronto.edu/~fritz/absps/momentum.pdf）
